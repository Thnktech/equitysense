Documentation

🚀 Getting Started

  1. Get an API Key: Visit Alpha Vantage to get a free key.
  2. Connect: Paste the key into the input box on the bottom left sidebar and click "Connect".
  3. Add Stocks: Go to the Portfolio tab and click "Add Position". Enter the ticker (e.g., AAPL), your average price, and share count.

📊 Portfolio Management

  • Export: Downloads a `.json` file of your current portfolio and settings. Use this to backup your data.
  • Import (Merge): Adds stocks from a JSON file to your existing list. useful if combining lists.
  • Import (Replace): Wipes your current list and replaces it with the file data.
  • Update Prices: Click the "Update Prices" button in the Portfolio header. Note: Free API keys allow 5 calls per minute. The app automatically queues requests to prevent errors.

🧠 User Classification (Profile)

Go to the "Profile & Health" tab to take the diagnostic quiz. This classifies you into one of 15 investor types (e.g., Risk Minimizer, Long-Term Compounder).

Why this matters: Your profile determines the "Overweight" threshold. For example, a Concentrator is allowed 25% in one stock, while a Diversifier is warned at 10%.

🛡️ Decision Engine

Buy Now?

Scans for stocks with Scores > 80 that are currently Underweight in your portfolio based on your profile limits.

Sell Now?

Scans for stocks with Scores < 60 (deteriorating fundamentals) OR stocks that are excellent but Overweight (need trimming).

Firewall

The master view. Shows the fundamental health score (0-100) for every holding, regardless of action needed.
