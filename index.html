User Manual

Disclaimer

This tool is for educational purposes only. It uses algorithmic rules based on user inputs. Not financial advice.

1. How to Start

  1. API: Get a free key from Alpha Vantage. Connect in the sidebar.
  2. Add: Go to Portfolio -> Add Position. Select your "Thesis Pillars" (Why you bought it).
  3. Update: Click "Update Prices" in Portfolio. This fetches the latest data for all tabs.

2. Why is "Exit" showing everything?

If you see errors or "Broken" statuses everywhere, click Update Prices in the Portfolio tab first. The Decision Engine needs fresh data (Growth, PE, Debt) to calculate scores. Without data, it assumes the worst (Score 0).
